API ENDPOINTS
================================================================================

POST /calculate
  Input: {"expression": "2 + 3 * 4"}
  Output: {"success": true, "result": 14, "expression": "2 + 3 * 4"}

GET /health
  Output: {"status": "ok"}

  Returns the calculator HTML interface
